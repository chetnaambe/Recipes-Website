# 🍔 Recipe Web App

A fully functional **Recipe Web Application** built using React, focused on real-world frontend development concepts and user interaction. This project demonstrates complete CRUD functionality along with authentication and modern UI practices.

---

## 🚀 Features

 **CRUD Operations**
  Create, Read, Update, and Delete recipes easily
 **Context API**
  Efficient global state management
 **Routing**
  Smooth navigation across multiple pages
 **Toast Notifications**
  Better user experience with instant feedback
 **Contact Form (Web3Forms)**
  Integrated contact functionality
 **Like Feature**
  Users can like their favorite recipes
 **Authentication**
  Login & Logout functionality using LocalStorage

---

## 🛠️ Tech Stack

* React JS
* JavaScript (ES6+)
* Context API
* React Router
* React Toastify
* Web3Forms
* LocalStorage

---

## 💡 What I Learned

* Building a complete **CRUD-based application**
* Managing state efficiently using Context API
* Handling routing and navigation in React
* Implementing authentication without backend
* Improving UI/UX with notifications and interactions
